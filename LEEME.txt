####################### CLASES ####################################

1. Clase de GrafoProbabilidad ---------------------------------

UBICADA EN = src/estructurasnolineales/GrafoProbabilidad.java

DESCRIPCION = Clase que permite realizar la estructura de un grafo de probabilidad.

METODOS = Tenemos varios metodos que nos permiten checar lo siguiente:
        - agregarVertice() - Este metodo agrega un verticeProbabilidad en el grafoProbabilidad.
        - agregarArista() - Este metodo agrega una arista en el grafo probabilidad utilizando como
                            argumento el verticeProbabilidad origen y el verticeProbabilidad destino.
        - buscarIndice() - Busca simplemente el indice de una info.
        - setProbabilidad() -Este metodo agrega la tabla de probabilidades al vertice del grafoProbabilidad.

2. Clase RedBayesiana --------------------

   UBICADA EN = src/RedesBayesianas/RedBayesiana.java

   DESCRIPCION =  Esta clase es la cual simula a la red de Bayes, el punto de esta clase es la de contener
                  al grafo de probabilidad y poder hacer las operaciones de guardar dichas probabilidades
                  en el mismo.

   METODOS = Tenemos 8 metodos principales, pero que mandan a llamar a otros que
             son privados para obtener ciertos datos, estos son:

               - agregarVariable(info): Agrega una variable a la red de Bayes.
               - agregarArista(origen, destino): Permite agregar una conexion entre dos variables.
               - agregarProbabilidades(variable): Dada una variable, es permitido buscarla y obtener su
                                                  tabla de probabilidades, para poder editar los valores
                                                  dentro de ella.
               - obtenerPadres(variable) [PRIVADO]: Metodo que dada una variable, se obtienen sus padres.
               - calcularProbabilidad(variable): Dada una variable de entrada, es posible obtener la probabilidad
                                                 de esta variable, preguntando por ciertos eventos de los padres.
               - buscarProbabilidad(variable): De una variable o vertice se puede obtener la tabla de
                                               probabilidad en tipo Matriz2
               - contarPadres(variable): Simplemente cuenta cuantos padres tiene un vertice.
               - imprimir(): solo imprime el grafo. Este metodo casi solo fue para pruebas.
               - calcularProbabilidadConjunta(): Este metodo calcula la probabilidad conjunta dada la cantidad de variables
                                                 seleccionada por el usuario, y las variables seleccionadas por el mismo.

3. Clase VerticeProbabilidad --------------------

   UBICADA EN = src/estructurasnolineales/auxiliares/VerticeProbabilidad.java

   DESCRIPCION =  Esta clase es para generar los vertices del grafo de probabilidad,
                  la diferencia de esta clase VerticeProbabilidad a la de Vertice es que
                  ademas de solo almacenar el indice y la info, el VerticeProbabilidad tambien
                  almacena una Matriz2Numerica que estamos usando como tabla de probabilidades
                  para ser usado y llenado con las probabilidades.

   METODOS = Tenemos varios metodos, estos son los getters y setters de los atributos
             indice, info y probabilidad.

4. Clase de Prueba: PruebaRedBayesiana --------------------------

    UBICADA EN = src/pruebas/PruebaRedBayesiana.java

    DESCRIPCIÓN = Esta clase es la clase que se uso para realizar pruebas respecto
                  a el uso de una Red Bayesiana.
